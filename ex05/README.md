# 第五回
## 負けるな！こうかとん　(ex05/fight_koukaton.py)
### ゲームの概要
- ex05/fight_kokaton.pyを実行すると，1600x900のスクリーンに草原が描画され，こうかとんを上下に移動させ敵に弾で倒すシューティングゲーム
- 敵のライフを0にするとゲームを終了する

### 操作方法
- 矢印キーの上下でこうかとんを移動させる

### 追加機能
- こうかとんのx座標を固定し、上下のみの移動にする
- 弾を実装
- 敵を実装
